## Командуудын жагсаалт
'git add *' **Бүх шинэ болон өөрчлөгдсөн файлуудыг стэйжилнэ**

'git add . *' **Бүх шинэ болон өөрчлөгдсөн файлуудыг стэйжилнэ**

'git add -A *' **Бүх шинэ болон өөрчлөгдсөн файлуудыг стэйжилнэ**

'git add [file.txt]' *' **file.txt файлыг стэйжилнэ**
